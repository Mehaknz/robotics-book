# Physical AI & Humanoid Robotics Capstone

This project contains the course specification for the Physical AI & Humanoid Robotics Capstone program.

## Project Structure

- `course_specification/`: Contains the main course specification document.
  - `Physical_AI_and_Humanoid_Robotics_Capstone.md`: The detailed course specification.
- `history/`: Contains the prompt history for the project.
- `.gemini/`: Contains configuration files for the Gemini CLI.
- `.specify/`: Contains templates and scripts for the Specify tool.

## Usage

This project is for documentation purposes. The main document is `course_specification/Physical_AI_and_Humanoid_Robotics_Capstone.md`.
