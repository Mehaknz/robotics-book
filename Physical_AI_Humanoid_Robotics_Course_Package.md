# Course Package for Physical AI & Humanoid Robotics

## Course Description
This course provides an in-depth exploration of Physical AI and humanoid robotics, focusing on the integration of artificial intelligence with physical systems. Students will learn about the design, development, and application of humanoid robots, utilizing various technologies such as ROS 2, Gazebo, Unity, and NVIDIA Isaac Sim. The course will also cover voice recognition and action execution using Whisper and GPT.

## Weekly Lecture Breakdown
### Week 1: Introduction to Physical AI and Humanoid Robotics
- Overview of Physical AI
- History and evolution of humanoid robotics

### Week 2: Fundamentals of Robotics
- Basic concepts in robotics
- Introduction to ROS 2

### Week 3: Gazebo Simulation
- Setting up Gazebo
- Basic simulation exercises

### Week 4: Unity for Robotics
- Introduction to Unity
- Creating robotic simulations in Unity

### Week 5: NVIDIA Isaac Sim
- Overview of Isaac Sim
- Integrating Isaac Sim with ROS 2

### Week 6: Voice Recognition Technologies
- Introduction to Whisper
- Implementing voice recognition in robotics

### Week 7: Action Execution with GPT
- Overview of GPT
- Integrating GPT with voice commands

### Week 8: Midterm Project Presentations
- Students present their midterm projects

### Week 9: Advanced Robotics Concepts
- Machine learning in robotics
- Sensor integration

### Week 10: Final Project Preparation
- Planning and designing the final capstone project

### Week 11: Capstone Project Development
- Hands-on development of capstone projects

### Week 12: Final Presentations and Course Wrap-up
- Presentation of final projects
- Course review and feedback

## Weekly Lab Assignments
### Week 1: Setting Up the Development Environment
- Install ROS 2 and Gazebo
- Basic ROS 2 commands

### Week 2: Building a Simple Robot Model
- Create a simple robot model in Gazebo

### Week 3: Simulating Robot Movement
- Implement basic movement commands in Gazebo

### Week 4: Unity Simulation Lab
- Create a robotic simulation in Unity

### Week 5: Isaac Sim Lab
- Develop a simple simulation using Isaac Sim

### Week 6: Voice Recognition Lab
- Implement voice commands using Whisper

### Week 7: Action Execution Lab
- Integrate GPT for action execution based on voice commands

### Week 8: Midterm Project Development
- Work on midterm project

### Week 9: Advanced Sensor Integration Lab
- Integrate sensors into the robot model

### Week 10: Capstone Project Planning Lab
- Outline and plan the capstone project

### Week 11: Capstone Project Development Lab
- Continue development of capstone project

### Week 12: Final Project Review Lab
- Review and finalize capstone projects

## Weekly Quizzes
- Quizzes will be administered at the end of each week to assess understanding of the material covered.

## Midterm Project
- Students will design and implement a midterm project that demonstrates their understanding of the concepts learned in the first half of the course. Projects will be presented in Week 8.

## Final Capstone Project Instructions
- The final capstone project will require students to design and develop a humanoid robot that can perform specific tasks using voice commands. Students will present their projects in Week 12.

## Hardware/Software Requirements
- **Hardware:**
  - A computer capable of running ROS 2 and Gazebo
  - Microphone for voice recognition
  - Optional: Robotics kit for hands-on projects

- **Software:**
  - ROS 2
  - Gazebo
  - Unity
  - NVIDIA Isaac Sim
  - Whisper and GPT libraries

## Integration Labs
- Labs will focus on integrating ROS 2 with Gazebo, Unity, and Isaac Sim, as well as implementing a voice-to-action pipeline using Whisper and GPT throughout the course.
