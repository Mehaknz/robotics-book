AI & Humanoid Robotics

Course Syllabus

Module 1: The Robotic Nervous System (ROS 2)

Focus: Middleware for robot control. Covers ROS 2 Nodes, Topics, Services, rclpy, and URDF for humanoids.

This module introduces the core concepts of the Robot Operating System (ROS 2), the middleware that enables communication and coordination between different parts of a robot. Students will learn to build and manage a robotic software system.

Module 2: The Digital Twin (Gazebo & Unity)

Focus: Physics simulation and environment building. Covers Gazebo physics, Unity rendering, and sensors like LiDAR, depth cameras, and IMUs.

Learn to create high-fidelity simulations of robots and their environments. This module covers the use of Gazebo for realistic physics and Unity for advanced rendering, along with the simulation of various sensors critical for robot perception.

Focus: Advanced perception and training. Covers Isaac Sim, synthetic data generation, Isaac ROS, VSLAM, and Nav2.

Dive into NVIDIA's powerful robotics platform. This module explores Isaac Sim for generating synthetic data to train AI models, and uses Isaac ROS for advanced perception tasks like Visual SLAM and autonomous navigation with Nav2.

Module 4: Vision-Language-Action (VLA)

Focus: LLMs + Robotics. Covers OpenAI Whisper for voice commands, LLM-based planning, and an autonomous humanoid capstone project.

Explore the cutting-edge intersection of large language models and robotics. Students will learn to use voice commands to control robots, leverage LLMs for complex task planning, and apply these skills in a final capstone project with an autonomous humanoid.
