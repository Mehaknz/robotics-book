# Course Package: Physical AI & Humanoid Robotics

## Course Description
This course provides an in-depth exploration of Physical AI and Humanoid Robotics, focusing on the integration of artificial intelligence with robotic systems. Students will learn about the design, programming, and application of humanoid robots, with a strong emphasis on practical labs and projects. The course will cover the use of ROS 2, Gazebo, Unity, and Isaac Sim for simulation and control of robotic systems, as well as the implementation of voice-to-action pipelines using Whisper and GPT.

## Weekly Lecture Breakdown
### Week 1: Introduction to Physical AI and Humanoid Robotics
- Overview of course objectives and expectations
- Introduction to Physical AI concepts
- History and evolution of humanoid robotics

### Week 2: Robotics Fundamentals
- Basic principles of robotics
- Kinematics and dynamics of robotic systems
- Introduction to sensors and actuators

### Week 3: ROS 2 Overview
- Introduction to ROS 2 architecture
- Setting up the ROS 2 environment
- Basic ROS 2 concepts: nodes, topics, services

### Week 4: Gazebo Simulation
- Introduction to Gazebo
- Creating and simulating robotic models in Gazebo
- Integrating ROS 2 with Gazebo

### Week 5: Unity for Robotics
- Introduction to Unity
- Building interactive simulations for robotics
- Integrating Unity with ROS 2

### Week 6: Isaac Sim
- Overview of Isaac Sim
- Setting up Isaac Sim for robotic simulations
- Integrating Isaac Sim with ROS 2 and Unity

### Week 7: Voice-to-Action Pipelines
- Introduction to voice recognition technologies
- Implementing Whisper for voice recognition
- Integrating GPT for action execution

### Week 8: Midterm Project Presentations
- Students present their midterm projects
- Peer feedback and discussions

### Week 9: Advanced Robotics Topics
- Machine learning in robotics
- Ethical considerations in AI and robotics
- Future trends in humanoid robotics

### Week 10: Capstone Project Preparation
- Guidelines for the final capstone project
- Team formation and project brainstorming
- Initial project proposals

### Week 11: Capstone Project Development
- Work on capstone projects
- Weekly check-ins and progress reports

### Week 12: Final Presentations and Course Wrap-Up
- Presentation of capstone projects
- Course review and feedback session

## Weekly Lab Assignments
- **Week 1:** Set up ROS 2 environment and basic node creation.
- **Week 2:** Build a simple robotic model and simulate basic movements.
- **Week 3:** Create a ROS 2 node that publishes sensor data.
- **Week 4:** Simulate a robotic arm in Gazebo and control it via ROS 2.
- **Week 5:** Develop a simple Unity application that interacts with a ROS 2 node.
- **Week 6:** Create a simulation in Isaac Sim and control it using ROS 2.
- **Week 7:** Implement a basic voice-to-action pipeline using Whisper and GPT.
- **Week 8:** Prepare for midterm project presentation.
- **Week 9:** Research and present on an advanced robotics topic.
- **Week 10:** Develop a project proposal for the capstone project.
- **Week 11:** Continue development on capstone project.
- **Week 12:** Finalize and present capstone project.

## Weekly Quizzes
- **Week 2:** Quiz on robotics fundamentals.
- **Week 4:** Quiz on ROS 2 concepts.
- **Week 6:** Quiz on simulation tools (Gazebo, Unity, Isaac Sim).
- **Week 8:** Midterm quiz covering Weeks 1-7.
- **Week 10:** Quiz on voice-to-action technologies.
- **Week 12:** Final quiz covering the entire course.

## Midterm Project
- Students will work in teams to design and implement a robotic system that performs a specific task. The project will require the integration of ROS 2, Gazebo, and a physical or simulated robot. Teams will present their projects in Week 8.

## Final Capstone Project Instructions
- The capstone project will involve the design and implementation of a humanoid robot capable of performing complex tasks. Students will work in teams to develop their projects, which must include:
  - A detailed project proposal
  - Integration of ROS 2, Gazebo, Unity, and Isaac Sim
  - Implementation of a voice-to-action pipeline using Whisper and GPT
  - A final presentation demonstrating the robot's capabilities

## Hardware/Software Requirements
- **Hardware:**
  - A computer capable of running ROS 2, Gazebo, Unity, and Isaac Sim
  - Optional: A physical robot for testing and implementation

- **Software:**
  - ROS 2 (latest version)
  - Gazebo (latest version)
  - Unity (latest version)
  - Isaac Sim (latest version)
  - Whisper and GPT libraries for voice-to-action integration

## Integration Labs
- Labs will focus on integrating ROS 2 with Gazebo, Unity, and Isaac Sim, as well as implementing voice-to-action pipelines using Whisper and GPT. Students will work on practical assignments that reinforce the concepts learned in lectures and prepare them for their projects.
